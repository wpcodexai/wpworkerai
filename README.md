# AllyWorker

![AllyWorker — Give AI agents full control of WordPress](https://raw.githubusercontent.com/allyworker/allyworker/refs/heads/main/wpworker_github_banner.svg)

[AllyWorker](https://allyworker.com) is where AI meets WordPress. We build the bridge between modern AI agents and the world's most popular CMS — turning any WordPress site into a fully programmable environment that an AI can operate like a developer would.

Our flagship product, **AllyWorker**, exposes WordPress over the [Model Context Protocol (MCP)](https://modelcontextprotocol.io) so AI clients like Claude, Cursor, and VS Code Copilot can install plugins, modify themes, run PHP, query the database, and build custom functionality — all without touching a dashboard.

We believe the next generation of WordPress development isn't typed by humans. It's delegated to AI — with humans staying in control of what matters.

Built and maintained by **[Aminul Islam](mailto:aminul@allyworker.com)** — a WordPress developer on a mission to make AI a first-class citizen in the WordPress ecosystem.
